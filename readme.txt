This WordPress plugin integrates Contact Form 7 forms into Conversational Form.

== Description ==

This Free WordPress plugin integrates Contact Form 7 forms into Conversational Form.

If you have any issues or questions about this plugin then please create a support request and we will get back to you shortly.

Once the plugin is activated you will have the option to enable/disable the form integration.


== Installation ==

1. Sign in to your WordPress site as an administrator.
2. In the main menu go to Plugins > Add New.
3. Search for "Conversational Form Add-on For Contact Form 7" (without the quotes of course) and click install.
4. After the plugin is installed activate the plugin.
5. That's it. Go ahead and start creating forms using Contact Form 7 and you will see a tab called "Conversational Form Integration".
